CUSTOM CURSOR DESIGN

Implement a modern minimalist cursor.

Requirements:

Hide the default system cursor.

Create two elements:

1. Inner dot
   • size ~4px
   • solid white
   • follows cursor instantly

2. Outer ring
   • size ~20px
   • thin white border
   • slight transparency
   • follows cursor with smooth delay

Interactions:

When hovering clickable elements:
• outer ring expands slightly
• subtle glow appears
• animation must be smooth and lightweight

Use requestAnimationFrame or performant animation techniques.

The cursor should feel elegant and futuristic, not distracting.
